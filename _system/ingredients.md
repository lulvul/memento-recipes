# Master Ingredient Registry

The running list of every standardized ingredient. Two jobs:
1. **Canonical names** so the same thing is spelled the same way everywhere.
2. **Grocery aisle + density** so lists group by where you shop and conversions
   to grams stay consistent.

Add a row whenever a new recipe introduces an ingredient. Keep it sorted by aisle
then name. Densities are standard kitchen references (estimates, not sourced) -
use them to convert volume measures to grams at intake.

Aisles: `produce`, `meat`, `dairy`, `canned`, `dry-goods`, `baking`, `spices`,
`oils`, `condiments`, `frozen`, `alcohol`, `other`.
